# GRRAASP
This repository contains the LabVIEW software and documentation required to operate the Guided Reaching and Repositioning Adjustable Assessment Platform (GRRAASP), a modular and sensorized setup designed for the standardized neural, muscular, biomechanical, and behavioral assessment of reach-grasp-transport tasks

Folder GRRAASP_LabView contains:
- GRRAASP_main.vi contains the main file controlling the platform.
- Check File Path.vi is needed to set the file path to save logs of the platform.
